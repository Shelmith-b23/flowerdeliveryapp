## Quick Component Selection Guide

| I want to... | Use class... | Example |
|---|---|---|
| Create a button | `.btn-primary`, `.btn-secondary`, `.btn-danger`, `.btn-success` | `<button class="btn-primary">Save</button>` |
| Display a flower card | `.flower-card` | See product display section above |
| Show a grid of items | `.flower-grid` | Wraps multiple `.flower-card` elements |
| Create a form | `.auth-card`, `.auth-input-group`, `.auth-input` | Login/register pages |
| Build a dashboard | `.bdb-container` or `.fd-container` | Main user area |
| Show a modal | `.fd-modal-overlay`, `.fd-modal` | Popup dialogs |
| Container wrapper | `.container` | General content wrapper |
| Generic card | `.card` | Any card content |
